# TWMDatabase
Dual part Python/SQL setup that imports a CSV purchase order, extracts relevant data, and stores it into a PostGRES database.

Old build uploaded for demonstration purposes. Requires manual data entry for retail pricing and QD information, but the rest of the process is automated. Working on developing web scraping API tools to import this automatically off of the Total Wine website. Some fields are missing information, but the critical components used in field are present.
